Documentación sobre la Creación de la Página Web

Desarrollo de la Galería

Lo primero que hicimos fue investigar sobre nuestro sector de las mañanitas que fue el sector 4 Para entender mejor la historia del sector 4, realizamos una investigación y obtuvimos Consulta de paginas web sobre el sector: se busco información sobre la fundación del sector 4, datos y cambios urbanos. Vista en primera persona del sector : en nuestra nuestras visita, tomamos fotos de los lugares más representativos, parques, y centros comunitarios. Lugares emblemáticos: parques, murales que reflejan la identidad del barrio. história: Calles y estructuras con valor arquitectónico o histórico.

Creación de la página web

Utilizamos herramientas básicas pero efectivas como algunas que fueron: HTML y CSS para estructurar y diseñar la página web. GitHub Pages para alojar la página y hacerla accesible en línea.

ESTE PROCESO LLEVO VARIOS PASOS COMO:

Crear una estructura inicial para la galería. Diseñar el formato visual, priorizando un diseño limpio que destacara las fotografías. Subir las imágenes y organizar las secciones según las categorías definidas.

Desafíos

Fue un reto garantizar que la página funcionara correctamente en todos los navegadores y dispositivos. También enfrentamos dificultades al organizar el código y al subir las imágenes a GitHub. Logísticas: a uno de nuestro compañeros por poco le roban el teléfono y a otro le quitaron la cartera entonces esperamos almenos un 4 en nuestro proyecto
